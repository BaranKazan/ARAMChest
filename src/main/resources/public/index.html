Welcome to ARAMChest API

Please enter the summoner name and the region of the summoner in URL. Note that the spacebar is not reqruied when entering the summoner name.

/summonerName/{summonerName}/region/{region}

Region tag:

  • BRAZIL
  • EUROPE_NORTH_EAST
  • EUROPE_WEST
  • JAPAN
  • KOREA
  • LATIN_AMERICA_NORTH
  • LATIN_AMERICA_SOUTH
  • NORTH_AMERICA
  • OCEANIA
  • RUSSIA
  • TURKEY

Example: /summonerName/KinkyMonkey/region/EUROPE_WEST
